# CreatorDNA Frontend

The CreatorDNA frontend provides creators with a dashboard for onboarding, profile management, and daily content opportunities.

---

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Axios

---

## Features

### Authentication

- Registration
- Login
- JWT session management

### Creator Onboarding

Submit:

- TikTok handle
- Example videos

Generate:

- Creator DNA profile

### Daily Ideas

View personalized content opportunities generated by the platform.

### Profile Management

Review creator profile and DNA characteristics.

## Pages

| Route          | Purpose             |
| -------------- | ------------------- |
| /auth/login    | User Login          |
| /auth/register | User Registration   |
| /onboard       | Creator Onboarding  |
| /today         | Daily Content Ideas |
| /history       | Previous Ideas      |
| /profile       | Creator Profile     |

## Development

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Start Production Server

```bash
npm run start
```
